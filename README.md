Генератор случайных паролей с графическим интерфейсом (Tkinter):
Настройка пароля — пользователь задаёт длину (4–128 символов) и выбирает типы символов: цифры, буквы, спецсимволы.
Генерация — кнопка создаёт случайный пароль, гарантируя, что в нём будут все выбранные типы символов.
Копирование — готовый пароль можно скопировать в буфер обмена одной кнопкой.
История — все сгенерированные пароли сохраняются в таблицу с датой и длиной, автоматически загружаются из JSON-файла при запуске.
Управление историей — можно удалять отдельные записи или очищать всю историю.
Проверки — контролируется минимальная/максимальная длина и наличие хотя бы одного типа символов.
